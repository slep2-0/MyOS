◆ init_heap()

void init_heap ( void )

Initializes the Kernel's HEAP for dynamic memory allocation.

The function declares the globals for the heap current end, as well as setting up the free_list pointer. It first creates 1 initial 4KiB frame, maps it to be paged in virtual memory, and increases the heap current end by the frame size (4KiB), so that 1 starting page is allocated.

◆ zero_bss()

void zero_bss ( void )

Zeroes out the BSS section (garbage data that might have left there)

The function takes the start and end address of the .bss section (provided by the linker script), and iterates over each byte from the start address up to (but not including) the end address, writing zero to each location. This ensures all global/static variables without initializers are zeroed.
